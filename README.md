# Food Pande

A beautiful static landing page for a bakery-themed food website. This project showcases a responsive layout, modern design, and a clean user experience for presenting fresh baked goods and local specialties.

## ✨ What’s included

- Responsive hero section with call-to-action buttons
- Featured menu item cards
- About section with brand story
- Contact form and business details
- Mobile-friendly navigation menu

## 🛠️ Technologies used

- HTML
- CSS
- JavaScript

## 🚀 How to use

1. Open `index.html` in a web browser.
2. Or serve the folder with a local web server for best results.

## 📁 Files

- `index.html` — main landing page content
- `styles.css` — responsive styling and layout
- `script.js` — mobile menu toggling behavior

## 💡 Notes

This project is ideal for a portfolio sample, bakery landing page, or small food brand demo. Feel free to customize the colors, images, and content to match your own food business.
